Como usar

Abra o arquivo executável.
Ao ver a tela inicial , a primeira coisa a ser feita é o cadastro dos seus arquivos
csv's.
Para isso, voce terá que ir na tela de Cadastro de Log's e escolher a unica 
opção do menu.
Ao apertar o botão Open File irá abrir um diretório. Escolha o arquivo
que desejar e após isso aperte o botão gravar para que o arquivo seja salvo.
Será necessário para que gere visualização no gráfico, ao menos um arquivo
contendo usuários e outro arquivo de qualquer atividade que relacione uma
atividade com um usuário.
