Como usar:

	Abra o arquivo executável(denominado Histograma.jar).
	Com o programa aberto é necessario cadastrar os seus arquivos csv's, caso não tenha nenhum arquivo cadastrado

Como cadastrar arquivos:

	Para cadastrar os arquivos basta selecionar o menu Cadastro de Log's, depois selecionar a opção Logs
	Quando selecionar essa opção uma nova pagina será aperta, nela se deve apertar o botão Open File para selecionar
	um diretorio e escolher uma arquivo. Após escolher o arquivo basta apertar o botão gravar para que o arquivo seja salvo.

Como ver os dados:

	Para ver os dados basta selecionar a opção Relatorio no menu, e depois selecionar a opção Geral.
	Com isso é possivel ver todos os dados que foram cadastrados.

Como visualizar a arvore:

	Para ver a arvore basta selecionar a opção Visualizar no menu, e depois selecionar a opção Arvore.
	Com isso é possivel ver a arvore de usuarios.


Como visualizar o histograma:

	Para ver o histograma basta selecionar a opção Histograma no menu, e depois selecionar a opção Selecionar Usuario.
	Com isso irá abrir uma nova paginá onde deverá ser digitado id do usuario procurado.
	Caso o usuario procurado exista será exibido os histogramas de suas atividades.
	Obs:Será necessário para que gere visualização no gráfico, ao menos um arquivo
	contendo usuários e outro arquivo de qualquer atividade que relacione uma
	atividade com um usuário.
	
Como fechar o programa:
	Para sair do programa basta selecionar a opção Sair no menu e depois selecionar a opção Salvar e sair.
	Com isso o programa é encerrado com todos os dados salvos.
